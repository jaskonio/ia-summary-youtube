# Guide to use

## Prerequites

- Ollama
- Python

## Development

```bash
python -m venv .env
 .\.env\Scripts\Activate.ps1 
python .\main.py
```

- [vc_redist.x64.exe](https://learn.microsoft.com/es-es/cpp/windows/latest-supported-vc-redist?view=msvc-170#latest-microsoft-visual-c-redistributable-version)

```bash
choco install ffmpeg
```
